# 💼 Vindobona Pro: Job Search & Interview Prep Checklist (Vienna Edition)

This checklist tracks all the steps we will take together to prepare your applications, locate matching positions in Vienna, navigate the visa process, and master technical and HR interviews.

---

## 🔍 Pillar 1: Target Job Hunting & Curation
We will search active job boards and target specific high-value companies in Vienna.

- [ ] **Job Search Sweeps**
  - [ ] Search [karriere.at](https://www.karriere.at) for "Junior/Mid React" and "Node.js Developer" in Vienna.
  - [ ] Search [LinkedIn Jobs](https://www.linkedin.com) for "Cloud Developer" and "DevOps Azure" in Vienna.
  - [ ] Check [devjobs.at](https://devjobs.at) for local Viennese startup positions.
- [ ] **Corporate Career Portals**
  - [ ] Check **Erste Digital** (George platform team) for Junior frontend/backend openings.
  - [ ] Check **Raiffeisen Bank International (RBI)** for Cloud/Infrastructure positions.
  - [ ] Check **Bitpanda** for Node.js, React, or SRE roles.
  - [ ] Check **Siemens Österreich** for industrial IoT and Azure cloud roles.
  - [ ] Check **BAWAG Group** and **UniCredit Bank Austria** for full-stack developer vacancies.

## 📄 Pillar 2: Resume & Application Portfolio
Crafting high-impact materials that highlight your security, transaction, and DevOps skills.

- [ ] **Resume/CV (Lebenslauf) Creation**
  - [ ] Write and optimize English CV (focus on full-stack architecture, Docker, and Azure).
  - [ ] Write and optimize German CV (leveraging C1 German competency for local firms).

  - [ ] Draft an English cover letter template for international tech scale-ups (e.g., Bitpanda).
  - [ ] Draft a German cover letter template for traditional corporate roles (e.g., Siemens, BAWAG).
- [ ] **Portfolio Presentation**
  - [ ] Tailor your GitHub repository README to explain the architecture of *Vindobona Pro FinTech*.
  - [ ] Prepare direct links to your live deployed site on Azure/Vercel.

## 🇦🇹 Pillar 3: Red-White-Red Card (Rot-Weiß-Rot-Karte) Visa Checklist
Since Software Engineering is a shortage occupation (*Mangelberuf*), we will track the paperwork required for visa sponsorship.

- [ ] **Points Calculator Verification**
  - [ ] Validate points for age, professional experience, and academic degrees.
  - [ ] Claim points for C1 German and C1 English language certificates.
- [ ] **Document Preparation**
  - [ ] Prepare C1 German certification (e.g., ÖSD, Goethe-Institut).
  - [ ] Prepare C1 English certification (e.g., IELTS, TOEFL, or Cambridge).
  - [ ] Prepare university diplomas/transcripts (with certified German/English translations if necessary).
  - [ ] Prepare official employment reference letters from previous employers.

## 💬 Pillar 5: Mock Interview Training
Interactive interview practice sessions to build confidence.

- [ ] **Level 1: Junior Basics (Whiteboard & Logic)**
  - [ ] Session on JavaScript fundamentals (closures, scope, `const`/`let`, array methods).
  - [ ] Session on React fundamentals (props vs state, hook lifecycles).
  - [ ] German language conversational check-in.
- [ ] **Level 2: Mid-Level Backend & DB (Node, APIs, SQL)**
  - [ ] Session on REST API design, HTTP status codes, and security headers.
  - [ ] Session on SQL queries (indexes, joins, and foreign keys).
- [ ] **Level 3: Security & Enterprise (ACID, JWT, Cloud)**
  - [ ] Session on SQL Transactions, ACID compliance, and concurrency controls.
  - [ ] Session on security (CORS, JWT cookies, rate limiting, 2FA integration).
- [ ] **Level 4: DevOps & System Design (Docker, Azure)**
  - [ ] Session on Dockerfiles, container networking, and Kubernetes services.
  - [ ] Session on Azure cloud setups (Azure Key Vault, App Services, ACR).
  - [ ] Whiteboard system design: how to scale a transaction system to 100k requests/sec.
