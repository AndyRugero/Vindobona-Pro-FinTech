# Vindobona Pro Fintech 🏆

Ein hochmodernes, modulares Finanz-Dashboard, das mit Fokus auf sauberen Code, Performance und erstklassiges Design entwickelt wurde. 

Dieses Projekt ist eine professionelle Neugestaltung des ursprünglichen Vindobona-Sandbox-Konzepts und dient als fortgeschrittene Lernplattform für moderne Webtechnologien.

## 🚀 Funktionen

- **Zentrales Hauptbuch (Central Ledger):** Eine übersichtliche Darstellung aller Transaktionen mit semantischer Farbgebung (Einnahmen vs. Ausgaben).
- **Intelligentes Formular:** Hinzufügen von Transaktionen mit automatischer Kategorie-Vorhersage basierend auf dem Empfänger.
- **Echtzeit-Statistiken:** Visualisierung von monatlichem Einkommen, Ausgaben und Netto-Ersparnissen.
- **Modulare Architektur:** Ein sauber strukturierter Codebase mit strikter Trennung von UI-Komponenten, Logik (Hooks) und Schnittstellen (Interfaces).
- **Premium Design:** Ein modernes Dark-Mode-Interface mit flüssigen Animationen und responsiven Layouts.

## 🛠️ Tech Stack

- **Kern:** React 19 & TypeScript
- **Build-Tool:** Vite
- **Styling:** Vanilla CSS (Custom Properties, Flexbox & Grid)
- **State-Management:** Custom React Hooks (`useTransactions`)
- **Logik:** Dedizierte Services für Geschäftslogik (`TransactionService`)

## 📦 Installation und Start

Um das Projekt lokal zu starten, folgen Sie diesen Schritten:

1. Klonen Sie das Repository:
   ```bash
   git clone https://github.com/IHR_BENUTZERNAME/Vindobona-Pro-Fintech.git
   ```

2. Installieren Sie die Abhängigkeiten:
   ```bash
   npm install
   ```

3. Starten Sie den Entwicklungsserver:
   ```bash
   npm run dev
   ```

Das Dashboard ist nun unter `http://localhost:5173` erreichbar.
